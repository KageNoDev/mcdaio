mcdaio documentation
====================

**mcdaio** is a small Python toolkit for multi-criteria decision analysis
(MCDA), shipped together with a Streamlit-based GUI.

It supports four classic MCDA methods - MABAC, TOPSIS, VIKOR and ARAS -
along with helpers for normalization, rank-correlation metrics, and
leave-one-out sensitivity analysis.
